 Intégration Réseau

                

                

                    

                        
1. Configuration IP Statique

                        
Assurez-vous que ce serveur dispose d'une adresse IP statique sur votre réseau (ex: 192.168.1.2).

                        
                        
2. Configuration du Routeur (Box Internet)

                        
Pour que WebGuard puisse filtrer le trafic, il doit être défini comme la passerelle par défaut ou le serveur DNS de vos appareils.

                        
    
                            
  • Option A (Recommandée) : Désactivez le serveur DHCP de votre Box et activez celui de WebGuard (si disponible) ou configurez manuellement vos appareils.
    • 
                            
  • Option B (DNS Uniquement) : Changez le serveur DNS primaire de votre Box pour l'adresse IP de ce serveur. Cela permettra le blocage de publicités via Pi-hole.
    • 
                        


                        
3. Configuration des Clients

                        
Si vous ne changez pas la configuration du routeur, configurez chaque appareil :

                        

                            IP: 192.168.1.x

                            Masque: 255.255.255.0

                            Passerelle: [IP DE CE SERVEUR]

                            DNS: [IP DE CE SERVEUR]
